### zen::tools::quota::set

Sets disk quota for a specific user.

#### Example

```bash
zen tools quota set username 10G
```

#### Arguments

* **$1** (string): Username
* **$2** (string): Quota limit (e.g., '10G' for 10 Gigabytes, 'max' for maximum available)
